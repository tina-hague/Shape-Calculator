
### Shape Calculator
This calculator uses a scanner to recieve input and calulate the surface area, volume, and perimeter of a 2D or 3D shape with the dimensions provided.
Available shapes:
- regular polygon
-  circle
-  ellipse
-  rectangular pyramid
-  sphere
-  cylinder
-  cone
